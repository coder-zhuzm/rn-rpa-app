# RPA App - React Native 自动化应用

一个基于React Native开发的RPA（机器人流程自动化）应用，支持Android设备自动化操作。

## 🚀 快速开始

### 开发环境启动

```bash
# 1. 安装依赖
cd RPAApp && npm install

# 2. 启动Metro服务器
npm start

# 3. 启动Android应用
npm run android

# 4. 启动Web调试界面
npm run web-debug
```
